== Description ==

This is a **Mac OSX Menubar** like **WordPres** adminbar/toolbar theme, designed for Mac and WordPress lovers. Read more about [iBar]( https://linesh.com/projects/ibar/).

== Installation ==

1. Upload **iBar** folder to the `/wp-content/plugins/` directory or download it from the administration
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate in Wordpress Admin Pages and see the new, clean design!

== Frequently Asked Questions ==

= Does the plugin change Wordpress layout? =
Yes, this plugin will make the CSS Style adminbar/toolbar only.

= Is the plugin compatible with all WP versions? =

No, because Wordpress could change after each update the admin style, and the plugin is based on it. So if you want to be sure to see the correct style of **iBar** you need last Wordpress version.
